# Installation

1. Install <a target="_blank" href="https://cli.angular.io/"> ANGULAR CLI </a>
2. Clone or download this repo
3. npm install
4. ng serve --prod --aot=false --sourcemaps=false
5. open browser navigate to http://127.0.0.1:4200
